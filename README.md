# CandiDATOS 2026

Dashboard interactivo con datos declarados por los candidatos a las **Elecciones Generales del Perú 2026**, según información del [JNE — Voto Informado](https://votoinformado.jne.gob.pe).

## ¿Qué muestra?

- Ingresos promedio declarados por partido
- Candidatos con sentencias judiciales
- Distribución por sexo por partido
- Origen de ingresos (público vs privado)
- Años de experiencia laboral
- Rankings individuales de candidatos

## Datos

Fuente: JNE — Voto Informado. Datos al 1 de marzo de 2026.
